## üéØ Overview

This **10-day intensive training program** is designed for **lateral hires with development experience** who want to master modern mobile application development. The training covers both **iOS** and **Android** platforms with a focus on contemporary frameworks and best practices.

### Training Format
- **Morning Sessions (4 hours):** iOS Development
- **Afternoon Sessions (4 hours):** Android Development
- **Daily Duration:** 8 hours (9:00 AM - 6:00 PM)
- **Total Duration:** 10 Full Days

### Key Highlights
- ‚úÖ Modern frameworks: **SwiftUI** & **Jetpack Compose**
- ‚úÖ Industry-standard architectures: **MVVM**
- ‚úÖ Real-world projects: **Banking & Insurance domains**
- ‚úÖ Cloud integration: **Firebase**
- ‚úÖ Production-ready practices
- ‚úÖ Hands-on, project-based learning

## üîß Setup Instructions

### iOS Development Setup

#### Step 1: Install Xcode
```bash
# Download from Mac App Store or
xcode-select --install
```

#### Step 2: Install Dependencies
```bash
# Using Swift Package Manager (Recommended)
# Dependencies are managed in Xcode

# Or using CocoaPods
sudo gem install cocoapods
pod setup
```

#### Step 3: Clone Project Repository
```bash
git clone https://github.com/your-org/securebank-ios.git
cd securebank-ios
```

#### Step 4: Open in Xcode
```bash
open SecureBank.xcodeproj
# or if using CocoaPods
open SecureBank.xcworkspace
```

#### Step 5: Configure Firebase
1. Download `GoogleService-Info.plist` from Firebase Console
2. Add to project root
3. Configure Firebase in `SecureBankApp.swift`

#### Step 6: Build and Run
- Select target device/simulator
- Press `Cmd + R` to build and run

---

### Android Development Setup

#### Step 1: Install Android Studio
```bash
# Download from: https://developer.android.com/studio
# Install Android SDK (API 24+)
# Set up Android Emulator or connect physical device
```

#### Step 2: Configure Environment
```bash
# Add to ~/.zshrc or ~/.bashrc
export ANDROID_HOME=$HOME/Library/Android/sdk
export PATH=$PATH:$ANDROID_HOME/emulator
export PATH=$PATH:$ANDROID_HOME/tools
export PATH=$PATH:$ANDROID_HOME/tools/bin
export PATH=$PATH:$ANDROID_HOME/platform-tools
```

#### Step 3: Clone Project Repository
```bash
git clone https://github.com/your-org/policypro-android.git
cd policypro-android
```

#### Step 4: Open in Android Studio
- Open Android Studio
- File > Open > Select project directory
- Wait for Gradle sync

#### Step 5: Configure Firebase
1. Download `google-services.json` from Firebase Console
2. Place in `app/` directory
3. Ensure Google Services plugin is applied in `build.gradle`

#### Step 6: Build and Run
```bash
# Using command line
./gradlew assembleDebug
./gradlew installDebug

# Or use Android Studio Run button
```
